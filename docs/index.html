Cloud Networking Concepts

This simulator demonstrates key cloud networking concepts including:

# Example: BGP/OSPF Hybrid Cloud Router Configuration
name: "Cloud Router Configuration"
description: "Multi-protocol router for cloud infrastructure"

interfaces:
  - name: "public-eth0"
    ip_address: "203.0.113.1"
    subnet_mask: "255.255.255.252"
    is_up: true
  
  - name: "private-eth1"
    ip_address: "10.0.1.1"
    subnet_mask: "255.255.255.0"
    is_up: true

bgp:
  as_number: "65001"
  router_id: "10.0.1.1"
  neighbors:
    - ip: "203.0.113.2"
      as: "65002"

ospf:
  router_id: "10.0.1.1"
  area: "0.0.0.0"
  interfaces:
    - "private-eth1"

traffic_shaping:
  token_bucket:
    capacity: 1000000000  # 1GB
    rate: 100000000      # 100MB/s
  wfq_classes:
    - class_id: 1
      weight: 10  # High priority for control traffic
    - class_id: 2
      weight: 5   # Medium priority for data traffic
    - class_id: 3
      weight: 1   # Low priority for bulk transfers

Quick Start

# Clone the repository
git clone https://github.com/yourusername/router-simulator.git
cd router-simulator

# Install dependencies
sudo apt-get install build-essential cmake libyaml-cpp-dev libpcap-dev
sudo apt-get install frr frr-dev

# Build the project
mkdir build && cd build
cmake ..
make -j$(nproc)

# Run tests
./router_test

# Start the simulator
sudo ./router_sim

# Load a scenario
load scenario scenarios/basic_router.yaml
