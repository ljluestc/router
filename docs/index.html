Quick Start

# Clone the repository
git clone https://github.com/yourusername/router-sim.git
cd router-sim

# Build the project
mkdir build && cd build
cmake ..
make -j$(nproc)

# Run with basic configuration
./router_sim -c ../examples/basic_router.yaml -i

# Execute test scenarios
./router_sim -s ../examples/test_scenarios.yaml
Get Started View Documentation GitHub Repository

Example Configuration

# Router Configuration
router_id: 1.1.1.1
hostname: production-router
enable_bgp: true
enable_ospf: true
as_number: 65001

# Interface Configuration
interfaces:
  - name: eth0
    ip_address: 192.168.1.1
    subnet_mask: 255.255.255.0
    bandwidth_mbps: 1000
    
# Traffic Shaping
traffic_shaping:
  eth0:
    rate_bps: 100000000  # 100 Mbps
    enable_wfq: true
    num_queues: 8
    
# Network Impairments
impairments:
  eth0:
    enable_delay: true
    delay_ms: 50
    enable_loss: true
    loss_percent: 1.0

  • Protocol Convergence Test: Failover and recovery scenarios
  • Load Balancing Test: Multi-path routing validation
